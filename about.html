About

This tool visualizes several algorithms which construct plane geometric spanners with bounded degree and stretch factor on a given set of two-dimensional points in the Euclidean plane. The properties of each algorithm are shown in the table below.

Directions

  1. An example point set is given. However, the points can be modified by clicking on the EDIT POINTS button at the bottom of the options pane.
  2. When you are satisfied with the point set, select an algorithm from the Algorithms select box.
  3. If you would like to only perform up to a certain step of the algorithm, select it from the step dropdown menu.
  4. Finally, select any additional options for the selected algorithm, if desired.
  5. Click GO.
  6. To run another algorithm on the same point set, return to step 2. The existing edges will be removed automatically.
